Why You Need SEO Ninja

Search engine optimization is critical for any website to surive and without having it done properly, it is going to be tough for your website to rank well in Google or other search egine result pages. With power of this plugin you'll be able to make your site more search engine and social friendly and boost your rankings.

How SEO Ninja Works

On installation, it creates database fields to store configurations and single posts meta. It also creates a robots.txt file to protect numerious directories from indexing. It inserts appropriate tags on all pages and improves overall website SEO.

SEO Ninja Configurations

This plugin gives admin control over its functionality. You can have a look at below settings to make this plugin according to your needs.

  1. Website Configurations > General Settings > Meta Title
    • Value: String of your choice
    • Default: Website Default Title

    Meta Title or SEO Title is the title that you set specifically for search engines. Type your seo title here in that field and make sure it is between 40 - 70 characters

  2. Website Configurations > General Settings > Meta Description
    • Value: String of your choice
    • Default: Website Default Description

    Meta Description or SEO Description is the description that you set specifically for search engines. Type your seo description here in that field and make sure it is between 120 - 156 characters

  3. Website Configurations > General Settings > Meta Keywords
    • Value: Comma (,) sepearated keywords

    Even though meta keywords is deprecated, there are still some search engines out there that care about it. Meta keywords are simply keywords that define your website. Add 4 + keywords in this field.

  4. Website Configurations > General Settings > Your Name
    • Value: You Real Name

    Search engines like to make sure that websites appearing at top of results are most trusted ones and one way to gain more trust in eyes of Google is telling that a real person is running the website

  5. Website Configurations > General Settings > Type
    • Value: Company / Person
    • Default: Person

    Search engines also have different indexing parameters based on if a website is being run by an organization. Select option that matches your situation

  • Website Configurations > General Settings > Single Item Meta
    • Value: Yes / No
    • Default: Yes

    Decide if you would like to give each post (video and photos only) a custom seo title etc.

  • Website Configurations > General Settings > Daily SEO Mail
    • Value: Yes / No
    • Default: Yes

    If you select yes, you will get a daily email with your current rankings

  • Website Configurations > Webmaster Tools > Google Verification ID
    • Value:Verification id given by Google

    Verify your site in Google Analytics or Webmaster tools using this field

  • Website Configurations > Webmaster Tools > Alexa Verification ID
    • Value:Verification id given by Alexa

    Verify your site in Alexa using this field

  • Website Configurations > Webmaster Tools > Bing Verification ID
    • Value:Verification id given by Bing

    Verify your site in bing Webmaster tools using this field

  • Website Configurations > Index Settings > Index Videos
    • Value:Yes / No

    Selecting 'No' will discourage search engines from indexing your website videos section

  • Website Configurations > Index Settings > Index Photos
    • Value:Yes / No

    Selecting 'No' will discourage search engines from indexing your website photos section

  • Website Configurations > Index Settings > Index Channels
    • Value:Yes / No

    Selecting 'No' will discourage search engines from indexing your website channels section

  • Sitemap Settings > General Settings > Submit Sitemap
    • Value:Action button

    Pushing this button will submit your website sitemap to Google and Bing
